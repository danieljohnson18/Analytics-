Artificial Intelligence Platform For Analytics

Analytics engine

With hundreds of data source connectors, simply add in various datasets and start analysing there is no limit to the data we can monitor and analyse.

Context Engine

The Context Engine is an NLP systems used to map user requests with the processed output of the Analytics Engine and entities in the knowledge graph. It also connects with external systems to identify contextual datasets.

Cognitive engine

The Cognitive Engine is a self-learning system that harnesses the capabilities of the Analytics Engine, Neural Network and the Context Engine to mimic the reasoning process of the brain. It is the powerhouse of the [AI] Platform.

Neural networks

Using powerful Neural Network libraries such as Tensorflow, the Neural Network is a deep learning system used by the regression algorithms of the Analytics Engine and Natural Language Processing system of the Context Engine.

Integrate any data source


With connectors to any data source via APIs, webhooks or CSV files there is no limit to the data we can analyse. You can add in various datasets to start training the AI Platform for analysis and insight.
